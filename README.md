# AI Dev Team Playground

This repository is an open-source playground designed to experiment with
AI-assisted software development.

## Goals
- Use AI tools to implement real features
- Review AI-generated code with AI reviewers
- Maintain human control and high code quality

## Non-Goals
- No production guarantees
- No proprietary code

## Tech Direction
- Language: TBD
- Focus: small, well-scoped features
- Strong emphasis on tests and clarity

## Contribution Rules (Important)
- Small pull requests
- Clear commit messages
- Tests required where applicable
- AI-generated code must be reviewed

## AI Usage Transparency
This project openly uses AI tools for coding and review.
All AI contributions are reviewed by humans before merge.
